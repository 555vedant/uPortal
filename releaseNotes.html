This is a file for notes specific to uPortal 2.6.0 release candidate 2. Part of the point of a release candidate release is to make a particular snapshot of progress evailable for review and evaluation, so what is known up front and could have been included in these release notes will be less than what is discovered about the release candidate post release.

Therefore, please see this release's wiki page for more information.

Requirements:

  • JDK 1.5 or later
  • Tomcat 5.x (5.5.x preferred). Other servlet containers may work. YMMV.
  • Database (Oracle, Postgres, MySQL are commonly used in production; these and HSQLDB are commonly used in development; specific vendors and versions are supported via declaration in the DbLoader utility; if you want to use a different database you'll need to declare its eccentricities there.)

What changed for this release

Issues resolved for this release are enumerated on the wiki page.

Known issues affecting this release

There are many known bugs in this release. You can query the JA-SIG JIRA system for known issues affecting this release, including issues discovered after this release went to press. Search for issues with a "Matches Versions" of this release (2.6.0 M2).

Please make known any issues you discover with this release.
