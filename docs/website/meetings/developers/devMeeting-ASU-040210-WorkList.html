uPortal Work List

The following to-do items were identified at the ASU uPortal Developers meeting February 9-10, 2004.

A Portlet-to-Channel adapter will definitely be part of uPortal 2.3, but the rest of the features are not scheduled for a particular release. The names listed next to some items indicate those meeting attendees that expressed interest in working on a particular features or set of features.
