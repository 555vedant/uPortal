How to Contribute

JA-SIG maintains a clearinghouse of Java projects, documents, procedures and tools that are available to share. All users wishing to contribute to the uPortal project must first register with the JA-SIG Clearinghouse. The login and password you set here will be the login and password you will use for accessing CVS.

We have created a generic account on our server for logging in and creating the SSH tunnel. The instructions for logging into this account will be sent to you upon request.

You will then want to review the Developer Information under the Documentation section of this site for details on setting up your SSH and CVS clients, reviewing the uPortal object model, and more.

JA-SIG Conferences

The Java in Administration Special Interest Group (JA-SIG) is an independent organization designed to increase the flow of information between educational institutions and companies involved in the development of administrative applications using Java technology. JA-SIG hosts a conference twice a year.
