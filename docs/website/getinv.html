How to Contribute

JA-SIG maintains a clearinghouse of Java projects, documents, procedures and tools that are available to share. All users wishing to contribute to the uPortal project must first register with the JA-SIG Clearinghouse. The login and password you set here will be the login and password you will use for accessing CVS.

We have created a generic account on our server for loging in and creating the SSH tunnel. The instructions for logging into this account will be sent to you upon request.
