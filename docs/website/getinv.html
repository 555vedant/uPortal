uPortal Contributors

Many have contributed to the uPortal project. A partial list of contributers and their main contributions to the uPortal project can be found here. Also listed are contributing organizations.

How to Contribute

The JA-SIG Clearinghouse is maintained by JA-SIG for sharing Java projects, documents, procedures, and tools. By following the link above, potential contributors can establish an account, review existing projects, and offer their own code for others to use. If you wish to contribute to the uPortal Framework itself, you should first subscribe to the uPortal Framework Developers mailing list as outlined above and discuss your contributions with the other developers. Code patches can be submitted via uPortal Bugzilla.

You will also want to review the Developer Information under the Documentation section of this site for details on setting up CVS access, reviewing the uPortal object model, and more.

Developer Meetings

uPortal developers get together twice a year. Information about uPortal developer meetings, a meeting schedule, and past meeting minutes can be found here.

JA-SIG Conferences

The Java in Administration Special Interest Group (JA-SIG) is an independent organization designed to increase the flow of information between educational institutions and companies involved in the development of administrative applications using Java technology. JA-SIG hosts a conference twice a year.
