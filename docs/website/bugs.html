Reporting Bugs

The uPortal project is using Bugzilla for reporting bugs and enhancement requests. At this time, the Bugzilla site is reviewed at the uPortal developer meeting which are presently occurring about once a month.

As ownership for the various project development areas is established, Bugzilla will be monitored regularly by the project task owner.
