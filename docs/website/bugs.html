Reporting Bugs

As of Dec 17,2004 the the uPortal project is using the Atlassian Jira project for reporting Issues (bugs and enhancement requests). Issues logged into Jira are tracked and reported in the uPortal Wiki". The following Jira projects have been created:

  • All Projects
  • uPortal
  • uPortal3
  • uPortal Documentation
  • Central Authentication Services (CAS)
