uPortal Tools

 

DbLoader/DbUnload

 DbLoader and DbUnload are tools which for creating, populating, and unloading the database tables used by uPortal
 

DbTest

 DbTest is a tool for unloading database properties and support capabilities.
 

md5passwd

 md5passwd is a tool for creating encrypted passwords for users in the uPortal database.
 

DeleteUser

 DeleteUser is a tool for deleting users from the uPortal database.
 

RegisterChannelType

 RegisterChannelType is a tool for registering new channel types in the uPortal database.
 

RegisterStyleSheet

 RegisterStyleShee is a tool for adding, deleting, and modifying stylesheets for uPortal.
