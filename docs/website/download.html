Download

Users wanting to quickly setup and demonstrate a uPortal site may download the quick-start distribution of uPortal.  This distribution provides a "drop it in and go" configuration of uPortal.  The intent is to make it easier for potential content developers to quickly get uPortal up and running and allow them to begin building content. The builds include a working configuration of the Tomcat Servlet Engine and HypersonicSQL database.

Users ready to deploy uPortal in a production environment should download the uPortal-only distribution containing only uPortal source code and configuration files (no servlet container or database). Use of this distribution requires knowledge of J2EE environments as well as the Ant tool. A servlet engine and relational database must be available.

The source code is included with each distribution. However for access to the latest code, visit the uPortal Source Code Repository.

See Getting Involved and the Documentation sections of this site for more information about creating content for uPortal.
