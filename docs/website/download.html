Download

The uPortal builds provide a "drop it in and go" configuration of uPortal.  The intent is to make it easier for potential content developers to quickly get uPortal up and running and allow them to begin building content. The builds include a working configuration of the Tomcat Servlet Engine and HypersonicSQL database.

The uPortal developers at Yale have put these builds together and you can download them using the links below.
 

Release Status Description
uPortal 2.0 "Arvon" Developer - Alpha Current development version. Feature incomplete. New architecture use XML, XSL and XSLT.
uPortal 1.5 "Destin" Stable - Current Latest stable release of uPortal.  Includes all the features of 1.0, support for Roles
uPortal 1.0  "Monterey" Stable - Old Production stable release of uPortal 1.0.
It is recommended that you download and implement uPortal 1.5 "Destin"

So what about Plymouth?  The "Plymouth" distribution released around Thanksgiving 2000, was an interim build created between uPortal 1.0 (Monterey) and uPortal 1.5 (Destin).  It was never really intended for general release and has been removed from distribution.
 

Users wanting to setup and demonstrate a uPortal site should download uPortal 1.5 "Destin".  Content created for the "Destin" release can be ported to uPortal 2 as long as certain programming consideration are adhered to.  See Getting Involved and the Documentation sections of this site for more information about creating content for uPortal.

The source code is included with each distribution, however for access to the latest code, visit the Source Code Repository for source updates.
