Maven configuration for uPortal 2.x has been moved to this experimental documentation folder because it is not being
actively maintained.  The release engineer did not use Maven for this build and dependency changes have been made for this 
build that are not reflected in this Maven configuration. 

Likely, if uPortal is to use Maven, it should use some flavor of Maven 2.

Extensive and thoroughgoing effort has been applied to use Maven 2 to build uPortal 3.
